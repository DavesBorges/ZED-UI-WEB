### Prerequisites

	-NODEJS 10 or superior
	-NPM 6 or superior
	-PHP 5 or superior

### Installing

	1 - git clone https://github.com/ZEDOSPROJECT/ZED-UI-WEB.git

	2a - Backend:
	  cd Backend; ./startBackend.sh

	2b - FrontEnd
	 cd Frontend;npm install ;npm start

	3b - FrontEnd Electron
	 cd Frontend;npm run electron
